¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de una página web mediante una serie de elementos o etiquetas que le indican al navegador cómo mostrar el contenido.

Estructura Básica

Un documento HTML básico tiene la siguiente estructura:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    <h1>Mi primera página web</h1>
    <p>Este es un párrafo en mi página web.</p>
</body>
</html>

Explicación de los elementos:

  • <!DOCTYPE html>: Declara el tipo de documento como HTML5.
  • <html>: El elemento raíz que contiene todo el documento HTML.
  • <head>: Contiene metadatos sobre el documento, como el título, enlaces a CSS, etc.
  • <meta charset="UTF-8">: Define la codificación de caracteres del documento.
  • <title>: Define el título de la página que aparece en la pestaña del navegador.
  • <body>: Contiene todo el contenido visible de la página web.

Elementos Básicos

Encabezados

HTML ofrece seis niveles de encabezados, desde <h1> (el más importante) hasta <h6> (el menos importante).

<h1>Encabezado de nivel 1</h1>
<h2>Encabezado de nivel 2</h2>
<h3>Encabezado de nivel 3</h3>
<h4>Encabezado de nivel 4</h4>
<h5>Encabezado de nivel 5</h5>
<h6>Encabezado de nivel 6</h6>

Párrafos

Los párrafos se definen con la etiqueta <p>.

<p>Este es un párrafo de texto.</p>
<p>Este es otro párrafo de texto.</p>

Enlaces

Los enlaces se crean con la etiqueta <a> y el atributo href.

<a href="https://www.ejemplo.com">Visita nuestro sitio web</a>

Imágenes

Las imágenes se insertan con la etiqueta <img> y los atributos src y alt.

<img src="imagen.jpg" alt="Descripción de la imagen">

Listas

Listas no ordenadas

Se crean con la etiqueta <ul> y cada elemento de la lista con <li>.

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Resultado:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Listas ordenadas

Se crean con la etiqueta <ol> y cada elemento de la lista con <li>.

<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>

Resultado:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Tablas

Las tablas se crean con las etiquetas <table>, <tr> (fila), <th> (encabezado) y <td> (celda).

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Fila 1, Celda 1</td>
        <td>Fila 1, Celda 2</td>
    </tr>
    <tr>
        <td>Fila 2, Celda 1</td>
        <td>Fila 2, Celda 2</td>
    </tr>
</table>

Resultado:

Encabezado 1 Encabezado 2
Fila 1, Celda 1 Fila 1, Celda 2
Fila 2, Celda 1 Fila 2, Celda 2

Formularios

Los formularios se crean con la etiqueta <form> y pueden contener diferentes tipos de elementos de entrada.

<form action="/procesar" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre"><br>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    
    <label for="mensaje">Mensaje:</label>
    <textarea id="mensaje" name="mensaje"></textarea><br>
    
    <input type="submit" value="Enviar">
</form>

Elementos Semánticos

HTML5 introdujo elementos semánticos que dan significado a las diferentes partes de una página web:

  • <header>: Define el encabezado de una página o sección.
  • <nav>: Define un conjunto de enlaces de navegación.
  • <main>: Define el contenido principal de la página.
  • <section>: Define una sección en un documento.
  • <article>: Define un contenido independiente y autónomo.
  • <aside>: Define contenido relacionado pero no esencial.
  • <footer>: Define el pie de página de una página o sección.

Consejos y Recursos

Consejos para principiantes

  • Siempre cierra tus etiquetas HTML correctamente.
  • Usa la indentación para hacer tu código más legible.
  • Valida tu HTML con herramientas como el Validador de W3C.
  • Aprende CSS para dar estilo a tus páginas HTML.
  • Practica creando pequeños proyectos para mejorar tus habilidades.

Recursos útiles

  • MDN Web Docs: Documentación completa sobre HTML.
  • W3Schools: Tutoriales y referencias de HTML.
  • CodePen: Plataforma para experimentar con HTML, CSS y JavaScript.
  • GitHub: Comparte y colabora en proyectos de código.
